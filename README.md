# 🎮 CartNest Gaming

A modern gaming store web app built using ASP.NET Core MVC.

## 🚀 Features
- User registration & login
- Admin portal
- Game categories
- Clean UI with modern design
- Session-based authentication

## 🛠️ Tech Stack
- ASP.NET Core MVC
- SQL Server
- Entity Framework Core
- HTML, CSS
